ShuffleForge — Перемешиватель списков на 7 языках
ShuffleForge — коллекция из семи независимых реализаций универсального перемешивателя списков. Каждая версия работает на своём языке программирования и предлагает расширенные возможности для случайного перемешивания элементов с контролем воспроизводимости, несколькими алгоритмами и экспортом результатов.

✨ Уникальные возможности (общие для всех версий)
🎯 Несколько алгоритмов перемешивания:

Fisher-Yates (классический, оптимальный)

Сортировка случайным ключом (с использованием sort с произвольным компаратором)

Перетасовка с использованием встроенной функции (где есть)

🔢 Поддержка различных типов данных: числа, строки, смешанные списки.

🎲 Управление случайностью:

Возможность задать seed для воспроизводимости результатов.

Режим истинной случайности (без seed).

📊 Статистика перемешивания: оценка случайности (энтропия, количество инверсий).

💾 Экспорт/импорт списков в JSON и CSV.

🔄 Генерация случайных списков для тестирования.

📋 Пакетный режим: перемешивание нескольких списков из файла.

🖥️ Интерфейсы:

Командная строка (CLI) с богатыми опциями.

Интерактивный режим (ввод списка, выбор алгоритма).

Веб-интерфейс в PHP и JavaScript.
